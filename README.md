# DynamicDict

Dynamic rank-1 dictionary estimation for fMRI functional connectivity (FC), using the PNC dataset. Presented at OHBM 2023.

# Synopsis

Connectivity dictionary entries have higher average connectivities for children compared to older adolescents.

<img width="600px" src='https://github.com/aorliche/DynamicDict/blob/3cb5f6da217a4613d1a53fdddc9d9cbd9a951faf/images/Connectivity_Components_Graphs.png'>

# Visualization of Dictionary Entries

Qualitatively different dictionary entries are correlated with young children compared to older adolescents, and with males vs females. 

<img width="600px" src='https://github.com/aorliche/DynamicDict/blob/3cb5f6da217a4613d1a53fdddc9d9cbd9a951faf/images/Correlation_Figure.png'>

# Phenotype Correlation with Dictionary Entries

Age is most highly correlated, followed by race, then sex, then achievement as measured by WRAT score.

<img width="600px" src='https://github.com/aorliche/DynamicDict/blob/3cb5f6da217a4613d1a53fdddc9d9cbd9a951faf/images/Correlation_Magnitude.png'>

# Age Prediction

Dynamic dictionary entry weights are slightly better at predicting age than PCA or kSVD, but not better than the full FC matrix.

<img width="600px" src='https://github.com/aorliche/DynamicDict/blob/3cb5f6da217a4613d1a53fdddc9d9cbd9a951faf/images/Age_Prediction_Figure.png'>
